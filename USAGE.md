# Using dotfiles

## 1. Setup

```shell
git clone https://github.com/lindhe/dotfiles.git ~/git/lindhe/dotfiles
~/git/lindhe/dotfiles/setup.sh
```

## 2. Install dependencies

```shell
sudo apt install \
    neovim tree zsh
```

### GUI extras

```shell
sudo apt install \
    dunst i3 j4-dmenu-desktop playerctl rofi
```
